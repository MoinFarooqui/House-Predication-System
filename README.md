# 🏠 House Prediction System

A **Flask-based Machine Learning Web Application** that predicts **house prices** using a dataset from Kaggle.  
This project combines the power of **Python, Flask, Pandas, NumPy, HTML, and CSS** to deliver an easy-to-use interface for users to estimate housing prices based on input features.

🔗 GitHub Repository: [House Prediction System](https://github.com/MoinFarooqui/House-Predication-System)

---

## ✨ Features
- 🔮 Predicts house prices based on input features  
- 🖥️ Simple and interactive web interface  
- ⚡ Fast and lightweight using Flask  
- 📊 Data preprocessing with **Pandas & NumPy**  
- 🎨 Responsive frontend with HTML & CSS  

---

## 🛠️ Technologies Used
- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS  
- **Data Handling:** Pandas, NumPy  
- **Dataset:** Kaggle (House Prices Dataset)
